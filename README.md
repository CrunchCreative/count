# Count

Football betting research and pattern discovery — iOS + Android app.

## Stack

- **Runtime:** Node 20.19.4+ (pinned via `.nvmrc`), pnpm 10
- **Monorepo:** pnpm workspaces
- **Mobile app:** Expo SDK 54 + React Native 0.81 + TypeScript
- **Routing:** Expo Router (file-based)
- **Styling:** NativeWind (Tailwind for React Native)
- **Data:** Supabase
- **Builds & distribution:** EAS Build + EAS Submit

## Getting started

```sh
pnpm install
pnpm dev
pnpm ios
pnpm android
```

See `docs/` for product, design, and architecture documentation.
